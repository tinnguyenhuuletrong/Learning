# bun-play-trpc

To install dependencies:

```bash
bun install
```

To run:

```bash

# terminal 1
bun --watch src/server/server.ts

# terminal 2
bun run src/client/client.ts
```

This project was created using `bun init` in bun v1.1.30. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
