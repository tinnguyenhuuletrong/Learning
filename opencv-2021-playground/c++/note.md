## Macos

1. Install opencv

```sh
brew install opencv
```

2. Dlib

- Download source http://dlib.net/files/dlib-19.21.tar.bz2
- Guide here http://dlib.net/compile.html
- link by `make install`

3. nlohmann/json

```sh
brew tap nlohmann/json
brew install nlohmann-json
```

3. Cmake

```sh
cmake .
make # output -> bin
```
